# Adventureworks-Sales-Analytics-SQL-Server
An end-to-end business analytics project built using SQL Server and the AdventureWorksDW2025 data warehouse. This project transforms raw dimensional data into a consolidated analytical dataset and performs comprehensive business analysis using advanced SQL techniques.

The project covers the complete analytics workflow, including data preparation, analytical modeling, KPI reporting, customer insights, product performance analysis, and advanced analytics such as RFM segmentation, Pareto Analysis, ABC Classification, and Cohort Retention Analysis.

## Project Objectives

The primary objective of this project is to simulate real-world business reporting using SQL Server by transforming the AdventureWorksDW2025 data warehouse into an analytical dataset capable of answering business questions.

The project focuses on:

- Building a clean analytical dataset from dimensional tables
- Performing sales performance analysis
- Understanding customer purchasing behavior
- Evaluating product performance and profitability
- Applying advanced analytical techniques using SQL window functions
- Demonstrating business-oriented SQL development suitable for data analyst roles

 ## Dataset

**Database:** AdventureWorksDW2025

AdventureWorksDW2025 is Microsoft's sample data warehouse that models the operations of a fictional bicycle manufacturing company. It contains historical sales, customer, product, reseller, geography, and time-based information designed for analytical workloads.

The project primarily utilizes:

- FactInternetSales
- DimCustomer
- DimProduct
- DimProductSubcategory
- DimProductCategory
- DimGeography
